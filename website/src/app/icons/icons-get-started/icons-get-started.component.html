Clarity Addons Icons

Import

To include the Clarity icons follow the instructions from the Clarity Icons - Get Started Guide.

Before you can use an icon, you must import if in your current component.ts file

You can also import all available icons at once

Usage of clarity addons icons is as easy as use the cds-icons tag with the desired shape

Badges

The following clarity addons icons support badges:

  • customer-vip
  • repeat-repair
  • customer-waiting
  • replacement-vehicle

Usage:
For details on how to use the badge property see Clarity Design - Icons.

Add badges to further icons:
To add badge support for a Clarity Addons icon, the icon need to be extended from IconSvgString into IconShapeCollection in the shapes.ts file. For more details on how to implement it, either compare the currently supported icons with the non-supported icons or look into the Clarity Design source code.
